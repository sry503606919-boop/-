#🎯 스마트 랜덤 선택 도구 Ultimate

## 📌 프로젝트 소개
본 프로젝트는 HTML / CSS / JavaScript를 기반으로 개발된 웹 애플리케이션으로, 여러 선택지 중에서 무작위로 결정을 도와주는 도구입니다.  

사용자는 여러 옵션을 입력하기만 하면, 애니메이션 회전판을 통해 자동으로 하나의 결과를 선택할 수 있습니다.

---

## ✨ 주요 기능

- 🎡 Canvas 기반 동적 회전판
- 🎯 포인터를 통한 결과 표시
- ⏳ 감속 회전 애니메이션 (현실적인 효과)
- 🔊 사운드 효과 지원
- 🌙 다크 모드 (야간 모드)
- 🌍 다국어 지원 (中文 / English / 한국어)
- 💾 로컬 저장소(LocalStorage)를 통한 기록 저장

---

## 🛠 기술 스택

- HTML5  
- CSS3 (그라데이션 및 UI 디자인)  
- JavaScript (Vanilla JS)  
- Canvas API (회전판 구현)  
- LocalStorage (데이터 저장)

## 🎯 智能随机决策工具 Ultimate

## 📌 项目简介
本项目是一个基于 HTML / CSS / JavaScript 开发的网页应用，用于帮助用户在多个选项中进行随机决策。

用户只需输入选项，即可通过动画转盘随机选出结果。

---

## ✨ 功能特点

- 🎡 动态转盘（Canvas实现）
- 🎯 指针指示结果
- ⏳ 减速旋转动画（更真实体验）
- 🔊 音效反馈
- 🌙 夜间模式
- 🌍 多语言支持（中文 / English / 한국어）
- 💾 本地历史记录（localStorage）

---

## 🛠 技术栈

- HTML5
- CSS3（渐变 + UI设计）
- JavaScript（原生）
- Canvas API（绘制转盘）
- LocalStorage（数据存储）
